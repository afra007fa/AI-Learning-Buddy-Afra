# AI Learning Buddy – Afra

## Overview
Afra is a friendly and encouraging AI Learning Buddy designed to help beginners learn Machine Learning concepts through simple explanations, real-world examples, interactive questions, quizzes, and personalized feedback.

This project was developed as part of the **AI EMPOW(H)ER Capstone Project by Infosys Springboard**.

## Features
- Beginner-friendly Machine Learning explanations
- Real-world examples and analogies
- Interactive quizzes with answers
- Personalized feedback
- Step-by-step learning approach
- Streamlit-based web interface

## Technologies Used
- Python
- Google Colab
- Streamlit
- ngrok
- Google Gemini API

## Project Structure
- `AI_Learning_Buddy.ipynb` – Google Colab notebook containing the project implementation
- `README.md` – Project documentation

**Name:** Afra

The objective of this project is to create an interactive AI-powered learning assistant that makes Machine Learning concepts accessible and engaging for beginners.
